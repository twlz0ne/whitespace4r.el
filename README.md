Minor mode to show whitespace for selected region.

![](./screenshot.png) 

## Installation

```elisp
(quelpa '(whitespace4r :fetcher github
                       :repo "twlz0ne/whitespace4r.el"
                       :files ("whitespace4r.el")))
```

## Usage

`M-x whitespace4r-mode`
